# ☕ OfCourse Coffee Web App

A modern coffee shop web application that allows users to explore menus, reserve tables, and order food online. The project focuses on user experience, clean UI, and real-world functionality.

---

## 🚀 Features

- 🔐 User Authentication (Sign up / Login)
- 🍽️ Table Reservation System
- 🛒 Food Ordering System
- 📰 Daily News Integration
- 💬 Daily Quote Display
- 📱 Responsive UI for different screen sizes

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- HTML, Tailwind css, JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Other**
- REST APIs
- Git & GitHub

---

## 📌 How It Works

- Users can create an account and log in securely
- Browse menu items and place food orders
- Reserve tables in advance
- View daily quotes and latest news updates
- All data is managed through backend APIs and stored in MongoDB
